RealRoots(coeff)
Signature: (coeff:float []) -> float []

Return the real roots of the supplied polynomial, that is stored with ascending coefficients: coeff[0] + coeff[1] x + coeff[2] x^2 + ... + coeff[n-1] x^(n-1) Note that double and triple roots are returned as repeated values. WARNING: currently only polynomials up to the 4th order can be solved.
RealRootsNormed(coeff)
Signature: (coeff:float []) -> float []

Return the real roots of the supplied normalized polynomial, that is stored with ascending coefficients: coeff[0] + coeff[1] x + coeff[2] x^2 + coeff[3] x^3 ... + x^n The last coefficient with value 1.0 must not be part of the supplied array. Note that double and triple roots are returned as repeated values. WARNING: currently only polynomials up to the 4th order can be solved.
RealRootsOf(a, b, c)
Signature: (a:float * b:float * c:float) -> Pair<float>

Return real roots of the equation ax^2 + bx + c = 0. Double roots are returned as a pair of identical values. If only one (linear) roots exists, it is stored in the first entry and the second entry is NaN. If no real roots exists, then both entries are NaN.
RealRootsOf(a, b, c, d)
Signature: (a:float * b:float * c:float * d:float) -> Triple<float>

Return real roots of the equation: a x^3 + b x^2 + c x + d = 0. Double and triple solutions are returned as replicated values. Imaginary and non existing solutions are returned as NaNs.
RealRootsOf(a, b, c, d, e)
Signature: (a:float * b:float * c:float * d:float * e:float) -> Quad<float>

Return real roots of: a x^4 + b x^3 + c x^2 + d x + e = 0 Double and triple solutions are returned as replicated values. Imaginary and non existing solutions are returned as NaNs.
RealRootsOfDepressed(p, q)
Signature: (p:float * q:float) -> Triple<float>

Return real roots of the equation: x^3 + p x + q = 0 Double and triple solutions are returned as replicated values. Imaginary and non existing solutions are returned as NaNs.
RealRootsOfNormed(p, q)
Signature: (p:float * q:float) -> Pair<float>

Return real roots of the equation x^2 + px + q = 0. Double roots are returned as a pair of identical values. If no real roots exists, then both entries are NaN.
RealRootsOfNormed(c2, c1, c0)
Signature: (c2:float * c1:float * c0:float) -> Triple<float>

Return real roots of the equation: x^3 + c2 x^2 + c1 x + c0 = 0 Double and triple solutions are returned as replicated values. Imaginary and non existing solutions are returned as NaNs.
RealRootsOfNormed(c3, c2, c1, c0)
Signature: (c3:float * c2:float * c1:float * c0:float) -> Quad<float>

Return real roots of: x^4 + c3 x^3 + c2 x^2 + c1 x + c0 = 0. Double and triple solutions are returned as replicated values. Imaginary and non existing solutions are returned as NaNs.
WithoutDoubleRoots(a, epsilon)
Signature: (a:float [] * epsilon:float) -> float []

Returns a copy of an array of roots without any double roots with an absolute difference that is smaller than the supplied epsilon. Roots with odd multiplicity will be left as single roots.
