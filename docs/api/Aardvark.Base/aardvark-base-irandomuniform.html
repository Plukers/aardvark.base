Aardvark.Base

IRandomUniform

This interface enforces a common API for random number generators.

Instance members

Instance memberDescription
GeneratesFullDoubles
Signature: bool
Modifiers: abstract

Returns true if the doubles generated by this random generator contain 52 random mantissa bits.
RandomBits
Signature: int
Modifiers: abstract

Returns the number of random bits that the generator delivers. This many bits are actually random in the doubles returned by .
ReSeed(seed)
Signature: seed:int -> unit
Modifiers: abstract

Reinitializes the random generator with the specified seed.
UniformDouble()
Signature: unit -> float
Modifiers: abstract

Returns a uniformly distributed double in the half-open interval [0.0, 1.0). Note, that only RandomBits bits are guaranteed to be random.
UniformDoubleClosed()
Signature: unit -> float
Modifiers: abstract

Returns a uniformly distributed double in the closed interval [0.0, 1.0]. Note, that only RandomBits bits are guaranteed to be random.
UniformDoubleOpen()
Signature: unit -> float
Modifiers: abstract

Returns a uniformly distributed double in the open interval (0.0, 1.0). Note, that only RandomBits bits are guaranteed to be random.
UniformFloat()
Signature: unit -> float32
Modifiers: abstract

UniformFloatClosed()
Signature: unit -> float32
Modifiers: abstract

Returns a uniformly distributed float in the closed interval [0.0f, 1.0f].
UniformFloatOpen()
Signature: unit -> float32
Modifiers: abstract

Returns a uniformly distributed float in the open interval (0.0f, 1.0f).
UniformInt()
Signature: unit -> int

UniformLong()

UniformUInt()

Returns a uniformly distributed integer in the interval [0, 2^32-1].
UniformULong()
Signature: unit -> uint64
Modifiers: abstract

Returns a uniformly distributed integer in the interval [0, 2^64-1].
