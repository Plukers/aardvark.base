FilterKernel

Convolution filter kernels.

Static members

Static memberDescription
Create1DKernel(radius, filterKernelFun)
Signature: (radius:int64 * filterKernelFun:Func<float32,float32>) -> Vector<float32>

Create a 1D filter kernel Vector populated by filterKernelFun, with size S = 2 * radius + 1 and anchor in the center at index 0. Functions to use as filterKernelFun are available in FilterKernelFun.
Create2DKernel(radius, filterKernelFun)
Signature: (radius:int64 * filterKernelFun:Func<float32,float32,float32>) -> Matrix<float32>

Create a 2D filter kernel matrix populated in accordance with filterKernelFun, with size (SX, SY) = (2 radius + 1, 2 radius + 1) and anchor in the center at index (0, 0). Functions to use as filterKernelFun are available in FilterKernelFun.
Create2DKernel(kernelX, kernelY)
Signature: (kernelX:Vector<float32> * kernelY:Vector<float32>) -> Matrix<float32>

Create a 2D kernel from two 1D kernels via outer product.

Swapping the two 1D kernel parameters gives the transposed 2D kernel.
