extractMaximalWhere condition t
Signature: condition:('a -> bool) -> t:Tree<'a> -> 'a option
Type parameters: 'a

finds and removes the maximal value from the tree for which the given condition holds. it is assumed that this condition is transitive with respect to the given comparison function. returns the (optional) value found in the tree. Example: extractMaximalWhere (fun v -> v < 10) gives the maximal value in the tree being smaller than 10. Runtime: O(log(N))
